ゲーム理論

Game theory provides a mathematical basis for modeling and analyzing a multi-agent system, which involves interactions among two or more self-interested and autonomous decision-makers, or agents. This course addresses basic game models, their solution concepts, and an introduction to evolutionary game theory.

生産管理技術Ⅰ

本講義では，現代の工業生産における生産管理の概要を把握した上で，在庫管理，生産計画，MRP，JIT生産，生産スケジューリング，ならびにそれらを支える情報システムの基礎などについて学ぶ．
