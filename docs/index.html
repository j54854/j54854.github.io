Welcome

青山学院大学 理工学部 経営システム工学科 集合知システム研究室（水山・杉之内 研究室）のウェブサイトにようこそ！

本研究室では，製造，物流，サービス提供などのシステム（広義の生産システム）は「複数の人と機械（計算機）の知的協働」によって動いていると考え，そのメカニズムの解明や支援手法の開発を目指して研究を行っています．
